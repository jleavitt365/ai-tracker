ⓘ Program Wave Guide

The AI introduction program rolls out department by department in three stages. Each stage gets a different level of IT support and pace. Here is what each one means.
Pilot
Pilot Wave — First Movers
Highest IT support · Sets the template for all other waves
Who
2–3 selected departments with strong champions and clear use cases
IT Support
High — IT Lead attends weekly check-ins and reviews all prompts
Goal
Prove the model works, capture 3+ reusable use cases, build the playbook
Timeline
Weeks 1–8 of the program
1
Nominated departments confirmed by IT Lead and dept head
2
60-minute intro meeting held before any access is granted
3
Department plan documented with 3 minimum use cases and baselines
4
Access granted — Tech Rep builds first working prompt workflow
5
Monthly progress review with IT Lead; results shared with leadership
Wave 2
Wave 2 — Expanding the Program
Moderate IT support · Learns from Pilot results
Who
Next tier of departments, onboarded after Pilot departments are activated
IT Support
Moderate — dept relies more on its own Tech Rep day-to-day
Goal
Reuse Pilot use cases, expand the library, reduce time-to-first-result
Timeline
Begins when at least 2 Pilot depts reach Step 4 (Activate)
1
Pilot results shared with Wave 2 Champions before their intro meeting
2
Use Case Library reviewed — Wave 2 starts with proven Pilot prompts
3
Same 5-step journey, faster pace — plan target is 5 days not 7
4
Tech Reps peer-connect with Pilot Tech Reps for prompt advice
Wave 3
Wave 3 — Full Rollout
Light-touch IT support · Departments largely self-serve
Who
Remaining departments — full organisation coverage
IT Support
Light — champions run onboarding using the established playbook
Goal
Complete org-wide coverage; Use Case Library at 25+ entries
Timeline
Begins when Wave 2 departments are activated
1
Champions use the playbook to run their own intro meetings
2
IT Lead reviews plans but does not attend all intro meetings
3
All departments contribute to and draw from the shared Use Case Library
4
Quarterly program review replaces monthly for mature departments
