About Pilot & Waves

Pilot: First 2–3 departments. High IT support, weekly check-ins, and building the shared use case library from scratch.

Wave 2: Next 4–6 departments. They benefit from pilot learnings and a library of proven prompts.

Wave 3: Remaining departments. Largely self-service using templates and prompts built by earlier waves.
